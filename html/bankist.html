Bankist App

The Bankist project is a web-based banking application developed using core web technologies, including HTML, CSS, and JavaScript. It leverages JavaScript for dynamic functionality, allowing users to log in, view their account balances, browse transaction history, and perform account-related actions. One notable technical feature is the use of the Date object to calculate and display transaction dates dynamically, providing users with real-time information about their account activities. The app also utilizes arrays and objects to efficiently store and manage user account data.
